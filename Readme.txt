The main goal of this educational project is to build a classification model that can predict the appropriate tariff for a customer based on their behavior, such as the number of minutes, gigabytes, and other usage characteristics of the telecommunications services.


The task is to develop a model that accurately determines whether the "Smart" or "Ultra" tariff should be offered to the customer. You will need to train the model using the provided data and evaluate its accuracy on a test dataset. The project's objective is to achieve an accuracy rate of at least 0.75.


Situation:


The mobile operator "XXX" has discovered that many customers are using outdated tariffs. They want to create a system capable of analyzing customer behavior and suggesting either the "Smart" or "Ultra" tariff.


We have access to data on customer behavior who have already switched to these tariffs. The task is to build a classification model that will select the appropriate tariff.


The aim is to build a model with the highest possible accuracy. The goal is to achieve a minimum accuracy rate of 0.75.






Основная цель этого учебного проекта - построить модель классификации, которая сможет предсказывать подходящий тариф для клиента на основе их поведения, таких как количество минут, гигабайт и других характеристик использования услуг связи. 


Задача состоит в том, чтобы модель с максимальной точностью определяла, стоит ли предложить клиенту тариф "Смарт" или "Ультра". Требуется обучить модель на предоставленных данных и проверить ее точность на тестовой выборке. Цель проекта состоит в том, чтобы достичь доли правильных ответов (accuracy) не менее 0.75.


Моделирование ситуации:


Оператор мобильной связи «ХХХ» выяснил: многие клиенты пользуются архивными тарифами. Они хотят построить систему, способную проанализировать поведение клиентов и предложить пользователям новый тариф: «Смарт» или «Ультра».


В нашем распоряжении данные о поведении клиентов, которые уже перешли на эти тарифы. Нужно построить модель для задачи классификации, которая выберет подходящий тариф.


Построить модель с максимально большим значением accuracy. Необходимо довести долю правильных ответов минимум до 0.75.
